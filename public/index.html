Cara membuat website simpel
dengan Next.js dan Tailwind CSS 10.30 ✓✓
X

Berikut langkah membuat website simpel dengan Next.js dan Tailwind CSS:

1. Buat Project Next.js

bashnpx create-next-app@latest xinn-website
cd xinn-website
npm run dev

2. Install Tailwind CSS

bashnpm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

3. Konfigurasi tailwind.config.js

jsmodule.exports = {
  content: ["./pages/**/*.{js,ts,jsx,tsx}",
  "./components/**/*.{js,ts,jsx,tsx}"],
  theme: { extend: {} },
  plugins: [],
}

4. Import Tailwind di globals.css

css@tailwind base;
@tailwind components;
@tailwind utilities;
